SortAndSearch

Die Applikation erstellt eine unsortierte Liste von Zahlen, die zufällig ausgewählt wurden.
Die Liste bekommt eine Größe, Minimum, Maximum und ob Duplikate erlaubt sind als Parameter.
Nach diesen Kriterien wird dann die Liste erstellt.

Sobald dann die List erstellt wurde, werden Bubble sort und Merge sort angewandt um die Liste zu sortieren.
Danach wird dann die Suche angezeigt und der User kann eine Zahl eingeben um diese in der Liste zu suchen.
Hier werden zum einem Binary Search und Jump Search angewandt. Falls die Zahl gefunden wurde, wird diese mit der Position dem User angegeben.
Ansonsten erscheint eine Fehler Meldung.
Auch hier wird die Zeit für beide Algorithmen gestoppt und im UI angezeigt.
